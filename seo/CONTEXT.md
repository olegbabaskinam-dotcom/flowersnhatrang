#### Как Claude делает СТАТЬЮ (метод, проверен)
1. `seo/articles/<slug>.json` — схема: id, slug, category, date, photo, related[3 slug товаров], per-lang {title,h1,meta,excerpt,intro,sections[{h2,html}],faq[[q,a]]}.
2. `import build_site as b` → `products=list(csv.DictReader(open('seo/products.csv')))` (НЕ load_products — её нет) → `arts=b.load_articles()` → `b.render_article(art,lang,products,arts)` → наложить CSS-патч (3 блока) → записать `blog/<slug>-{ru,en,ko}.html`.
3. **Метку категории на странице статьи локализовать руками** (генератор НЕ переводит): sed по строке «<Категория> · N min read/분량». EN/KO примеры: Повод→Occasion/특별한 날, Отель→Hotels/호텔.
4. Карточки в blog-{ru,en,ko}.html вставлять РУКАМИ, первой (новейшая сверху), с локализованной меткой категории.
5. sitemap.xml: +3 URL-блока (ru/en/ko + x-default hreflang). Валидировать `xmllint --noout sitemap.xml`.
6. registry.csv: status_ru/en/ko → done.

#### Фото статьи
- Карточки блога: `class="w-full h-48 object-cover"` (высота фикс) → нужен ГОРИЗОНТАЛЬНЫЙ кадр ~**1200×760** (thumb 600×380). Вертикальное фото в карточке выглядит обрезанным.
- Конверт: Pillow, ImageOps.exif_transpose, crop при необходимости, webp quality=82 method=6. Исходник класть в `img/_src/<тема>/`.
